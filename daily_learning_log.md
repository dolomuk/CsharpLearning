# 일일 학습 기록

### 학습한 주제
- C# 콘솔 애플리케이션 기본 구조 이해
- `dotnet` CLI를 사용한 프로젝트 생성 및 빌드
- 사용자 입력 처리 (`Console.ReadLine`) 및 데이터 검증 (`int.TryParse`)

### 주요 학습 내용
1. **C# 콘솔 애플리케이션 구조**
   - `Main` 메서드가 프로그램의 진입점임을 이해함.
   - `namespace`와 `class`의 역할을 학습.

2. **사용자 입력 처리**
   - `Console.ReadLine`을 사용해 사용자 입력을 받는 방법을 학습.
   - `int.TryParse`를 사용해 입력값을 정수로 변환하고 유효성을 검증.

3. **`dotnet` CLI 사용**
   - `dotnet new console` 명령어로 프로젝트를 생성.
   - `dotnet build` 명령어로 프로젝트를 빌드하고 실행.

4. **.md 이해하기**
    -  `MarkDown`파일이 무엇인지 그리고 어떠한 구조를 가진지 학습했음.
    -  앞으로 학습 내용은 이 .md 파일에 작성할거임.


### 어려웠던 점

- 사용자 입력값 검증 로직(`int.TryParse`)을 처음 접하며 약간의 혼란이 있었음.

    -   TryParse를 쓰는게 속도면에서도 안정성 면에서도 더 뛰어남

4. **간단한 계산기 프로그램**
